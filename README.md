# Room Booking — сайт бронирования комнаты с домашним кинотеатром

## Цель проекта
Создать лендинг для бронирования комнаты с домашним кинотеатром в центре Москвы.

## Пользователи
Клиенты могут:
- Ознакомиться с описанием комнаты и удобствами
- Просмотреть галерею фотографий
- Выбрать свободную дату и забронировать комнату
- Смотреть отзывы клиентов

## Основные сценарии
1. Клиент заходит на сайт.
2. Читает описание комнаты, смотрит удобства и отзывы.
3. Просматривает галерею.
4. Открывает форму бронирования и выбирает свободную дату (только 3 даты в неделю).
5. После бронирования дата отображается красным у всех пользователей.

## Архитектура
- **Frontend:** React + BEM для организации стилей
- **Backend:** Node.js + Express + SQLite
- **Компоненты:**
  - BookingForm — календарь с подсветкой занятых дат
  - Reviews — отзывы клиентов с аватарками
  - Features / Advantages — карточки удобств и преимуществ
  - Gallery — галерея с открытием полноразмерного фото
- **База данных:** SQLite хранит брони и обеспечивает проверку доступных дат.

## Принципы
- Все компоненты стилизованы по методологии **БЭМ**
- Чистая архитектура: модули изолированы, слабая связанность, каждый класс/компонент отвечает за одну задачу
- Принципы SOLID соблюдены (Single Responsibility, Open/Closed, и т.д.)

## Итоги
- Сделан адаптивный лендинг с 3 секциями: Hero, Booking, Gallery, Reviews, Footer.
- Реализована форма бронирования с проверкой занятых дат.
- Добавлены анимации hover для карточек удобств и отзывов.
- Использован BEM для всех стилей, проект легко масштабировать.
